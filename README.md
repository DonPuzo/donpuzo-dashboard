# 📊 AJSS Editorial Analytics Dashboard

A real-time, web-based analytics system designed to monitor journal performance, publication metrics, and editorial workflow efficiency for the **Alvan Journal of Social Sciences (AJSS)**.

---

## 🚀 Key Features

* 📈 Real-time tracking of submissions, accepted, and rejected manuscripts
* 📊 Automated computation of acceptance and rejection rates
* 📅 Monthly and annual publication trend analysis
* 🔄 Live data integration via Google Sheets API
* 🌐 Fully deployable via GitHub Pages for remote accessibility

---

## 🧠 Application Context

This system is developed to support:

* Academic journal management and editorial decision-making
* Research performance monitoring in higher institutions
* Evidence-based publishing strategies
* Procurement, governance, and policy analytics

---

## 🛠️ Technology Stack

* **Frontend:** HTML, CSS, JavaScript
* **Visualization:** Chart.js
* **Data Source:** Google Sheets API (JSON integration)
* **Deployment:** GitHub Pages
